# Indian Public School – Thakurpukur Website

## 🌐 Overview

This project is the official website developed for **Indian Public School, Thakurpukur** using the Laravel framework. The platform provides students, parents, and visitors with easy access to school information, announcements, academic details, and institutional updates through a modern and responsive web interface.

---

## ✨ Features

* 🏫 School information and academic details
* 📰 News & announcements management
* 🖼️ Gallery and event showcase
* 📄 Dynamic content management system
* 🔐 Secure admin panel
* 📱 Fully responsive design
* ⚡ Optimized performance using Laravel MVC architecture

---

## 🛠️ Tech Stack

* **Framework:** Laravel (PHP)
* **Backend:** PHP 8+
* **Database:** MySQL
* **Frontend:** HTML5, CSS3, JavaScript
* **Build Tool:** Vite
* **Server:** Apache / Nginx

---

## 📂 Project Structure

app/ → Application logic
resources/ → Blade templates & frontend assets
routes/ → Web routes
database/ → Migrations & seeders
public/ → Public assets
storage/ → Logs & cache files

## 🚀 Installation

1. Clone repository:

```bash
git clone https://github.com/your-username/indian-public-school-thakurpukur.git
```

2. Install dependencies:

```bash
composer install
npm install
```

3. Setup environment:

```bash
cp .env.example .env
php artisan key:generate
```

4. Configure database in `.env`.

5. Run migrations:

```bash
php artisan migrate
```

6. Start development server:

```bash
php artisan serve
npm run dev
```
